～開発中～

画面の最前面に選択した画像を常に表示します。

透明度とか位置とか変更できます。


使用例１：ゲームしながら同じ画面でメモ画像を見る際

使用例２：絵の練習での画像トレース



■機能

　言語設定

　テーマ設定

　画像をオーバーレイ表示
 
　　└画像位置設定
 
　　└画像透明度設定

 　メモオーバーレイ表示
 
　　└メモ位置設定
 
　　└ウィンドウ透明度・ウィンドウカラー・テキストカラー設定
