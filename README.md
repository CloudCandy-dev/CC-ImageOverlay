# CC-ImageOverlay

Windows用 イメージオーバーレイ & メモツール

## 概要
CC-ImageOverlayは、デスクトップ上に画像やテキストメモを半透明で表示できるツールです。
ゲームや作業中の参照用として最適です。

## 主な機能
- **画像オーバーレイ**: 好きな画像をデスクトップの最前面に表示
- **メモ機能**: シンプルなテキストメモを表示
- **リッチなUI**: グラスモーフィズム（すりガラス）デザインとモダンなアニメーション
- **多言語対応**: 日本語、英語、中国語、韓国語に対応

## 動作環境
- Windows 10 / 11
- .NET 10 Runtime (自己完結版を使用する場合は不要)

## 開発者向け情報
このプロジェクトはC# (WPF) / .NET 10で作成されています。

### ビルド方法
```bash
dotnet build
```
